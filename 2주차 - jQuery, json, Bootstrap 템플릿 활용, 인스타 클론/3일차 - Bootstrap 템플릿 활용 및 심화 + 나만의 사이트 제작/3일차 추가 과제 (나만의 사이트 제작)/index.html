Ming's Coding Playground(MCP)는
당신이 마음껏 코딩 테스트를 할 수 있는 장소를 제공합니다.
코딩에 도전해보고 싶을 땐 언제든지 들러주세요.

더 알아보기

무료로 제공되는 이 템플릿을 마음껏 수정해보세요!

물론 이 사이트 역시 StartBootstrap에서 무료로 다운로드 받은 자료를 수정한 버전입니다.
그렇지만 그 점은 별로 중요하지 않습니다.
개발 왕초보자인 제가 이렇게 만들었으니,
당신도 충분히 개발에 도전할 수 있다는 사실이 더 중요합니다.
